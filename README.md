# Quiz Application

## Description
A Python project that allows users to take a multiple-choice quiz and automatically calculates the score.

## Features
- Multiple-choice questions
- Score calculation at the end
- Console-based interface

## Technologies Used
- Python

## How to Run
1. Open the quiz_application.py file in Python IDE.
2. Run the program.
3. Answer the questions as displayed.
4. View your score at the end.
